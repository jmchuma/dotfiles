

Play all images in a direcotory

feh -D 5 -F -z directory
feh -D 5 -B black -S mtime -dr -.
feh -D5 -Borange -Smtime -dr.



COMMAND LINE OPTIONS

-B colour:
-Bcolour : background colour for transparet areas
-S mtime :
-Smtime  : sort by modified time

-D n:
-Dn : next image every n secs

-.  : fit image to window
-F  : full screen
-d  : show filename
-r  : recursively
-z  : randomize



KEYBINDINGS

h  : pause slideshow.
m  : show menu.
<  : rotate image 90 degrees counterclockwise.
>  : rotate image 90 degrees clockwise.
_  : flip image vertically.
|  : flip image horizontally.
